# Present Continuous 

## Use
* О действии, которое происходит в данный момент e. g. **)
О (с фразами )

## Forms

### Positives

#### Regular verbs
    I + V-ed/ -d (если глагол оканчивается на -е)
> 

#### Irregular verbs
    I + V2
> 

### Questions
    Did + I + V?
> 

    Where + did + I + V?
> 

### Negatives
    I + did not/ didn’t + V
